# Pedestrian-Tracking-for-Accident-Prevention
[차량비전시스템] YOLO, DeepSORT, EKF를 이용한 보행자 탐지, 추적 및 미래 경로 예측 시스템

## Installation
1.  Clone this repository into your `catkin_ws/src` directory.
2.  Install required Python packages:
    ```bash
    pip install -r requirements.txt
    ```
3.  Build the package:
    ```bash
    cd ~/catkin_ws
    catkin build pedestrian_tracking
    ```
